Estrutura Básica do HTML

Um HTML básico é composto por tags que marcam o conteúdo do arquivo de acordo com os padrões da Linguagem de Marcação de HiperTexto.
Veremos essas tags básicas e sua aplicação logo abaixo.

  • <!DOCTYPE html> - Esta declaração define que este documento é um documento HTML5
  • <html> - Este é o elemento raiz de uma página HTML
  • <head> - Este contém metainformações sobre a página HTML
  • <title> - Este especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página)
  • <body> - Este define o corpo do documento e é um recipiente para todo o conteúdo visível, como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

Todas essas tags precisam ser fechadas com uma barra e o nome da tag, após o conteúdo. Exceto a declaração do tipo do documento. Por exemplo a tag <body> deve ter a tag de fechamento </body> logo após a finalização dos dados que pertencem a essa marcação.

Dica! Caso você use o Visual Studio Code, você poderá utilizar o comando HTML:5 a fim de que o aplicativo crie automaticamente para você um padrão HTML 5 de inicialização, como o que segue abaixo.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

Formatação de Texto

As Tags para formatação de texto são muito utilizadas e geralmente precisam ser fechadas, para delimitar os caracteres do texto que sofrerão as alterações desejadas.

Segue abaixo algumas tags mais comuns como exemplo, já aplicadas no texto que as define.

  • <b> - Negrito sem qualquer importância
  • <strong> - Negrito com forte importância
  • <i> - Itálico
  • <em> - Itálico também, mas um leitor de tela pronunciará com ênfase.
  • <u> - Sublinhado, representando um texto não articulado com estilo diferente do texto normal.
  • <ins> - Sublinado, porém define um texto que foi inserido no documento.
  • <mark> - Marcado
  • <small> - Pequeno
  • <del> - Riscado
  • <sub> - Subscrito
  • <sup> - Sobrescrito

A tag <font> não é mais utilizada a partir do HTML5, embora essa tag pudesse ter a capacidade de mudar o estilo da fonte, o tamanho, e a cor do texto ela foi descontinuada. Deixando a função apenas para o CSS.

Listas HTML

No HTML temos três tipos de listas que podem ser marcadas para exibição. Dentre elas estão as Listas Não Ordenadas (Unordered HTML List), Listas Ordenadas (Ordered HTML List) e as Listas de Descrição (HTML Description Lists).

As listas Ordenadas e Não Ordenadas tem um arranjo parecido, mudando apenas a tag inicial e final, de acordo com a lista específica que se dejesa visualizar. Mantendo a semelhança enter a tag <li> usada para cada item.

Exemplo de Lista Ordenada, percebe-se que ela enumera a lista. (Tag <ol>)

  1. Cachorro
  2. Gato
  3. Papagaio

Código:

<ol>
    <li>Cachorro</li>
    <li>Gato</li>
    <li>Papagaio</li>
</ol>

Exemplo de Lista Não Ordenada, nesse caso a lista tem apenas símbolos em cada item. (Tag <ul>)

  • Cachorro
  • Gato
  • Papagaio

Código:

<ul>
    <li>Cachorro</li>
    <li>Gato</li>
    <li>Papagaio</li>
</ul>

Exemplo de Lista de Descrição, ela contem termos que são descritos um por um. (Tags <dl>, <dt>, <dh>)

Cachorro
Melhor amigo do homem
Gato
Adora ficar em lugares altos

Código:

<dl>
    <dt>Cachorro</dt>
    <dd>Melhor amigo do homem</dd>
    <dt>Gato</dt>
    <dd>Adora ficar em lugares altos</dd>
</dl>
